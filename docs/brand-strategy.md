# PennyForge — Brand Strategy

Naming, positioning, and voice recommendations for the PennyForge product: a compliance-first,
receipt-verified, community-scored local deal-intelligence platform for penny items and hidden
clearance. See `docs/product-spec.md` for the product itself and `docs/compliance.md` for the
data-sourcing rules every piece of copy below is written to respect.

Positioning anchor: **"Waze for hidden clearance: receipt-verified local deal intelligence, not
random Discord chaos."** The durable wedge is trust, proof, routing, ROI, community, and
compliance — never raw data volume, and never language that reads as exploitative.

---

## 1. Evaluating "PennyForge" as a name

**Strengths**

- "Penny" instantly signals the niche and matches the dominant naming convention among existing
  penny-shopping communities and tools (Penny Finder, Penny Central, Penny Puss, Krazy Coupon
  Lady's penny-list content) — high category recognition, low explanation cost.
- "Forge" reads as *craft, build, make real* — a well-worn developer/maker metaphor (SourceForge,
  CurseForge, Autodesk Forge) with no history of "forgery" confusion in those brands.
- The compound is currently unregistered as a US trademark (0 hits on a Trademarkia search).
- The name is already load-bearing throughout this codebase (schema comments, docs, README,
  product spec) — a rename has real switching cost.

**Risks (from live web research — see Appendix A for sources)**

| Risk | Finding | Severity |
|---|---|---|
| Domain availability | `pennyforge.com` has been registered since **2003-03-17** (not for sale, returns HTTP 503) | Medium |
| Recent squatting | `pennyforge.net` was registered **2025-02-26** and is a "launching soon" email-capture placeholder | Medium |
| Exact-name collision | An existing fintech product markets itself as **"PennyForge — Capital Clarity Studio"** (capital-planning software) at a live site | Medium |
| Handle collision | The GitHub handle `pennyforge` is taken by an unrelated hobbyist; a "The Penny Forge" Facebook page and a "Five Penny Forge" knife maker also use the string | Low–Medium |
| Trademark density | "Penny" is heavily claimed in software/finance classes: Penny AI (funded, active), Pennylane (accounting unicorn), PennyMac, a fresh April-2026 "PENNY" filing for AI financial software | Medium |
| Retailer-trademark adjacency | **PENNY Markt** is a well-known European discount-grocery chain (3,300+ stores) — a real conflict risk in the exact "retail/shopping" semantic space if PennyForge ever expands to the EU | Medium (EU-specific) |
| Connotation | "Forge" doubles as "to forge/fake" — for a brand whose *entire* differentiator is receipt-verified authenticity, "forged receipts" is an ironic own-goal a critic or headline writer could reach for | Low–Medium |

**Verdict:** PennyForge is **usable but contested**, not clear. No blocking US trademark was found,
but the .com is unavailable, a same-name product exists in an adjacent software category, and the
"forge/forgery" double meaning is a real (if minor) messaging risk for a trust-first brand. This
report is search-signal only — it is not a substitute for a professional USPTO/EUIPO clearance
search, which should happen before any paid marketing spend, trademark filing, or app-store
submission under this name.

## 5. Brand voice

### Voice principles

1. **Proof over hype.** Lead with evidence — receipts, tags, confirmations — never adrenaline.
   *Sounds like:* "Confirmed twice today, receipt attached — confidence 84." *Never:* "INSANE
   deal, RUN to your store NOW!!!"
2. **Competent neighbor, not deal-bro.** Plain words, specific numbers.
   *Sounds like:* "Worth the drive if you're within 15 miles — after gas, you're still ahead."
   *Never:* "Bro, this is basically free money."
3. **Honest about uncertainty.** Clearance moves fast; say so instead of overpromising.
   *Sounds like:* "Last confirmed yesterday. Prices vary by store and change fast." *Never:*
   "Guaranteed in stock at your local store!"
4. **Above-board, always.** Every find is ordinary shopping — an item on a shelf, bought at the
   register. *Sounds like:* "Every find is a first-hand, in-store report." *Never:* "The secret
   pricing system stores don't want you to know about."
5. **Local and human.** Data has a ZIP code and a name attached. *Sounds like:* "Reported by
   @maria_atl, confirmed by two shoppers near Marietta." *Never:* "Our proprietary data engine."
6. **Respectful of stores and staff.** Retailers are the venue, not the adversary. *Sounds like:*
   "Be kind to store employees." *Never:* "Get in and out before an associate notices."

### Vocabulary — say / don't say

| Say | Don't say |
|---|---|
| verified find | glitch deal, price glitch |
| unadvertised markdown / hidden clearance | secret deals stores are hiding |
| report a find | leak a deal, drop intel |
| confidence score (0–100) | sure thing, guaranteed deal |
| confirmed by shoppers | insider info, backdoor info |
| marked dead / no longer there | killed by the store, yanked |
| first-hand, in-store report | data feed, scraped listing |
| worth the drive / positive ROI after gas | run, don't walk / free money |
| trust score, earned through confirmed reports | power user status, deal god |
| purchased at the register like any other sale | beating the system, gaming the markdown |

### Tone across surfaces

- **App UI:** compact, factual, sentence case. Statuses state evidence and consequence, not
  emotion ("Suppressed — 2 dead votes outnumber confirms"). Buttons name the actual action
  ("Report a find," "Confirm," "Dead / gone"). Show the math (confidence breakdown, gas-cost ROI)
  instead of adjectives.
- **Notifications:** specific, honest about freshness, never manufactured urgency — product,
  store, confidence, recency, every time. One alert per find per day, and say so as a feature:
  "We won't ping you twice for the same find."
- **Marketing site:** warmer, proof-forward, longer sentences allowed — but every claim maps to a
  real mechanism (voting, scoring, receipts, routing). Contrast with chaos ("group-chat rumors")
  without naming or disparaging any specific community or competitor.
- **Community guidelines:** plain, firm, second person. Lead with the allowlist: what counts as a
  valid source (saw it, bought it, photographed the tag, public signage) and what never does
  (anything not personally witnessed in-store). Standing courtesy rule: buy at the register, leave
  shelves tidy, be kind to employees.

## 6. Homepage hero copy

*"Waze for hidden clearance" is internal positioning shorthand — Variant A carries the metaphor
(live, community-kept, ahead-of-you) without naming another brand in customer-facing copy.*

### Variant A — "Live map" (Waze-leaning)

- **Headline:** The live map of local clearance
- **Subheadline:** Shoppers ahead of you report the markdowns they found, confirm what's still
  there, and mark what's gone — so your trip counts.
- **Primary CTA:** See finds near me · **Secondary CTA:** How verification works
- **Bullets:**
  - A local feed of penny and clearance finds, filtered by state, store, and confidence
  - Confirm/dead voting keeps every lead current — stale finds disappear automatically
  - A route planner that ranks trips by expected value minus gas, and skips trips that lose money

### Variant B — "Trust and receipts" (proof-leaning)

- **Headline:** Deals with the receipts attached
- **Subheadline:** Every find is a first-hand, in-store report scored 0–100 on evidence, reporter
  track record, confirmations, and freshness. No rumors, no reposts.
- **Primary CTA:** Browse verified finds · **Secondary CTA:** Read our sourcing rules
- **Bullets:**
  - Receipts and shelf-tag photos score highest — see exactly why each lead scores what it does
  - First-hand reports only: no scraping, no private data, no recycled lists
  - Check any UPC, SKU, or product name before you drive

### Variant C — "Neighbors" (community-leaning)

- **Headline:** Your town's best finds, shared
- **Subheadline:** Join local shoppers who report real in-store markdowns, earn trust with every
  confirmed find, and get alerts when something good lands nearby.
- **Primary CTA:** Join your local feed · **Secondary CTA:** Report your first find
- **Bullets:**
  - Trust scores earned through confirmed reports — your word carries weight because you've been
    right before
  - Alerts for high-confidence finds near you, never more than one ping per find per day
  - A leaderboard that credits the contributors who keep your area's map alive

## 7. App-store-safe description

**App title tagline (≤30 chars):** `Verified local clearance finds` (30 chars)

**Short promo line (≤80 chars):** `Receipt-verified clearance finds near you, scored and mapped by real shoppers.` (79 chars)

**Full description (~320 words):**

> PennyForge is a community app for finding deeply discounted clearance items at nearby big-box
> stores. Every listing starts as a first-hand report from a shopper who was actually in the
> store, backed by a receipt, a shelf-tag photo, or an in-person observation.
>
> Each report earns a confidence score from 0 to 100 based on evidence strength, the reporter's
> track record, community confirmations, and how recently the item was seen. When a find stops
> panning out, shoppers mark it dead and it drops out of the feed — so you spend your time on
> items that are still on the shelf.
>
> **How it works**
> - Browse a local feed of clearance and penny-priced finds, filtered by state, store, and
>   confidence level.
> - Search any UPC, SKU, or product name to check a specific item before you drive.
> - Found something yourself? Submit a report with the price, store, and your evidence.
> - Confirm finds that are still there, and mark gone ones dead — the community keeps the map
>   honest.
> - Get alerts when high-confidence finds appear near you (never more than one alert per find per
>   day).
> - Use the route planner to see which trips are worth it: it ranks stores by expected value minus
>   round-trip gas cost and hides trips that would lose you money.
>
> **What PennyForge is not**
> - Not affiliated with, endorsed by, or connected to any retailer.
> - Not a scraper. We never collect data from retailer websites, internal systems, or third-party
>   feeds. Every data point comes from a person who stood in the store.
> - Not automation. PennyForge never purchases, reserves, or interacts with any store system on
>   your behalf.
> - Not a guarantee. Prices are community-reported, vary by store, and change quickly. Availability
>   is never promised.
>
> Clearance pricing is set by each store, and every find is purchased at the posted register price
> like any other sale. Be kind to store employees — they're the reason this hobby works.
>
> See what your neighbors have already found.

## 8. Compliance-friendly disclaimers

Grounded in `lib/compliance.ts` and the existing footer in `app/layout.tsx`. Items flagged
**[ATTORNEY REVIEW]** should get counsel sign-off before launch.

**1. General accuracy / YMMV**
> Deals on PennyForge are community reports, not guarantees. Every price you see here is a
> point-in-time observation made by a shopper in one specific store, on one specific day.
> Clearance and penny pricing varies store by store and can change within hours — an item that
> rang up at one price yesterday may ring up differently today, or may no longer be on the shelf
> at all. Confidence scores reflect community evidence (receipts, photos, confirmations), not a
> promise. PennyForge does not guarantee that any item will be available, or that it will ring up
> at the reported price, at any store. Always check the price at the register before you commit
> to a purchase.
> **[ATTORNEY REVIEW]:** add an express no-warranty clause and a limitation-of-liability
> cross-reference to the Terms of Service.

**2. No retailer affiliation**
> PennyForge is an independent, community-run service. We are not affiliated with, endorsed by,
> sponsored by, or connected to any retailer, including any retailer named in the app. Retailer and
> store names appear solely to identify the location where a shopper reported a find. All
> trademarks, brand names, and logos are the property of their respective owners, and their use
> here does not imply any partnership or endorsement.
> **[ATTORNEY REVIEW]:** confirm nominative-fair-use posture on retailer names before displaying
> anything beyond plain-text names.

**3. Data-sourcing statement (public trust statement)**
> Where our data comes from — and where it doesn't. Every deal on PennyForge comes from one place:
> a real shopper, standing in a real store. Our reports are first-hand observations — an item seen
> on a shelf, a purchase backed by a receipt, a photographed clearance tag, or publicly posted
> store signage. That's the complete list. We do not scrape retailer or third-party websites. We
> do not access retailers' internal systems, private endpoints, or employee tools. We do not
> ingest, repurchase, or repost data from other deal services. And we never use bots or automated
> tools to probe store inventory or prices. Our submission system enforces this: any report that
> isn't first-hand, in-store evidence is rejected before it's ever saved.

**4. Community conduct / honest reporting**
> Report what you saw. Nothing else.
> - Submit only finds you personally observed in-store: an item you saw on the shelf, a price you
>   saw at the register or on a tag, or a purchase you made yourself.
> - Receipts and photos are your proof. Reports backed by a receipt or shelf-tag photo earn higher
>   confidence and build your trust score.
> - Never fabricate, exaggerate, or repost someone else's find as your own, and never submit data
>   from scraped sites, other deal communities, or a retailer's internal systems.
> - Respect store staff and store policies. Be courteous, and don't ask employees to check
>   internal systems for you.
> - A store may decline to sell an item at a shelf or scanned price, or may pull it from sale
>   entirely — that is the store's right.

**5. App-store / website footer boilerplate**
> PennyForge is an independent community platform for sharing first-hand, in-store clearance
> finds. All prices and availability are user-reported, point-in-time observations; they vary by
> store, change frequently, and are not guaranteed. PennyForge is not affiliated with, endorsed
> by, or sponsored by any retailer; retailer names identify report locations only, and all
> trademarks belong to their owners. Stores set their own prices and may decline to honor a shelf
> or scanned price. Data comes exclusively from shopper reports — never from scraping, private
> systems, or third-party feeds. Use of PennyForge is subject to our Terms of Service and Privacy
> Policy.
> **[ATTORNEY REVIEW]:** this entire block, plus Terms/Privacy links, before app-store submission.

**6. In-app microcopy (condensed)**
- *Accuracy (product/lead pages):* "Community-reported find. Prices vary by store and change fast
  — always confirm at the register. Nothing here is guaranteed."
- *No affiliation (footer):* "PennyForge is independent and not affiliated with any retailer.
  Store names show where shoppers reported finds; trademarks belong to their owners."
- *Honest reporting (report-submission screen):* "Only report what you personally saw in this
  store — receipts and tag photos make your report count for more. Be kind to store staff; stores
  may decline to sell at a scanned price, and that's their call."

`docs/compliance.md`'s "Disclaimers shown to users" section calls for the accuracy disclaimer on
every pricing page plus store-discretion language — items 1, 4, and 6 satisfy that requirement.

## 9. SEO keyword clusters

Sampled from live search results (July 2026) — see Appendix A for sources.

1. **Penny list core** (informational, high recurring volume) — *dollar general penny list this
   week*, *how to find penny items at dollar general*, *penny shopping for beginners*. Winning
   angle: a living, dated "This Week's Community-Verified Penny List" hub, beating static blog
   lists on recency and proof.
2. **Hidden clearance & markdown schedules** (informational) — *walmart hidden clearance*, *target
   markdown schedule by day*, *what does .01 pricing mean*. Winning angle: evergreen "How Markdown
   Cycles Actually Work" explainers sourced from aggregated verified sightings.
3. **Apps like / alternatives to** (comparison, highest commercial intent) — *brickseek
   alternatives*, *is brickseek accurate*, *deal soldier vs brickseek*, *apps like brickseek*.
   Winning angle: honest comparison pages plus a "why receipt-verified beats scraped data" page —
   the compliance stance is itself a positioning wedge ("data retailers can't object to").
4. **Local intent** (local/transactional) — *clearance near me*, *penny items near me*, *walmart
   clearance [city/zip]*. Winning angle: programmatic local landing pages fed by geo-tagged
   community reports, plus a route-planner feature page carrying the "Waze" framing.
5. **Receipt-verification / trust** (informational, low volume, high conversion, brand-defining) —
   *are penny lists real*, *is penny shopping legal*, *deal soldier legit*. Winning angle: "Why
   Deal Data Is Usually Wrong (and How Receipt Verification Fixes It)" plus a legality/etiquette
   explainer.
6. **Couponing-community adjacent** (informational/community) — *penny shopping facebook groups*,
   *clearance discord servers*, *krazy coupon lady penny list*. Winning angle: "Tired of
   scroll-forever Discord and Facebook chaos?" migration-oriented content; court these communities
   as supply-side contributors, never mock them.
7. **Price-tag & scanner decoding** (informational, strong featured-snippet potential) — *walmart
   price ending .01 meaning*, *target clearance tag decode*. Winning angle: retailer-by-retailer
   visual decode guides funneling into "log your find in PennyForge."
8. **Observed "glitch/hack" demand — capture with compliant reframing only.** Real, high-volume
   queries exist (*walmart glitch deals*, *clearance hacks*). Strategy: never use
   glitch/hack/trick/secret-exploit language in titles, URLs, or copy. Capture the intent with
   educational reframes that rank on semantic proximity instead — "How penny items actually work
   (it's not a glitch)," "Why the app price differs from the shelf tag." Hard line: no content
   implying price-error exploitation or "before the store fixes it" urgency.

## 10. Final recommendation

**Keep "PennyForge" as the working name for now, but treat naming as still open pending legal
clearance — do not consider it locked.**

Reasoning:

- The build, schema, docs, and product spec already carry the name at real switching cost, and it
  matches the naming convention shoppers already recognize in this niche (Penny Finder, Penny
  Central, etc.) — that recognition has SEO and word-of-mouth value the alternatives would have to
  earn from zero.
- No US trademark currently blocks the name, but the collision picture (a taken `.com`, a
  same-name fintech product, a squatted `.net`, a taken GitHub handle) is real enough that this
  should not be treated as clear. **Commission a professional USPTO/EUIPO clearance search before
  any trademark filing, paid marketing spend, or app-store submission.**
- **Immediately secure what's still available**: `pennyforge.app` / `.io` (unverified — check
  directly), the iOS/Android app-store listing names, and social handles across
  Instagram/TikTok/X, before a rebrand decision is even needed.
- If clearance comes back blocked, or if a rebrand is chosen proactively to shed the
  forge/forgery double meaning, **ShelfProof** and **DealRadius** are the vetted fallbacks — both
  scored cleanly in this research, both encode a different half of the positioning (proof vs.
  local/routing), and both are short enough to survive an app icon.
- Whichever name ships, the brand voice, disclaimers, and SEO strategy in this document are name
  agnostic and should carry over unchanged — they were built from the compliance rules and shipped
  feature set, not from the name.
